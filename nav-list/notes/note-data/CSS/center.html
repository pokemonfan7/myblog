确定盒子宽高,利用负外边距

.parent{
position:relative;
width:200px;
height:200px;
border:1px solid #000;
}
.child{
position:absolute;
left:50%;
top:50%;
width:100px;
height:80px;
margin-top:-40px;
margin-left:-50px;
border:1px solid #000;
}

position & transform: translate(-50%, -50%) 实现块元素百分比下居中

.parent{position: relative;}
.child{
position: absolute;
top: 50%;
left: 50%;
transform:translate(-50%,-50%);
}

利用flexbox

.parent{
　　　　justify-content:center;

　　}

               height: 200px;
               position: absolute;
               top: 0;

               bottom: 0;
